== Description ==

Display your blog's PageRank. The WebInformationService PageRank plugin allows users to see the PageRank for your website.

WebInformationService is a free analysis tool for websites. We collect domain related data from various sources and condense, analyse and process it.

The WebInformationService PageRank button can also be generated purely as HTML without using this plugin. Just use the following link: http://www.webinformationservice.net

== Installation ==

1. Upload the webinformationservice-pagerank-button folder (unzipped) to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Appearance > Widgets' and drag the WebInformationService PageRank Widget to your sidebar
1. Configure the widget and press save
